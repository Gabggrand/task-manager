# Java Task Manager (Java Puro)

Sistema de gerenciamento de tarefas desenvolvido em **Java puro**, utilizando apenas recursos da linguagem e bibliotecas padrão do JDK.  
O projeto permite criar, listar, concluir, remover e salvar tarefas em arquivo, mantendo os dados entre execuções.

Este projeto foi desenvolvido com foco em aprendizado prático de:
- Programação Orientada a Objetos
- Manipulação de arquivos
- Estruturas de dados
- Lógica de programação
- Boas práticas em Java

---

## Funcionalidades

- Criar tarefas com título e categoria
- Listar todas as tarefas
- Marcar tarefas como concluídas
- Remover tarefas pelo ID
- Salvamento automático em arquivo
- Carregamento automático das tarefas ao iniciar o programa

---

## Tecnologias Utilizadas

- Java 17+
- IntelliJ IDEA
- Git e GitHub
